# CardInfo

Inspired by [CardInfoFindfer](https://github.com/KryptKode/CardInfoFinder)

# Get Started
Generate file *.g.dart, you can use this command on terminal :

```
flutter pub run build_runner build
```

## Features
- Splash screen
- Main screen
- Settings screen
- Manual input screen
- Card info screen

## Tech Stack
- Dart
- Provider
- Flutter BLoC
- Dio
- Get_it
- Auto_route
- Freezed
- Equtable
- JSON serializable
- Easy localization
- Shared preferences
